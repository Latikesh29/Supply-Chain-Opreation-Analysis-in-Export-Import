## 📖 Project Overview

This project presents an interactive **Power BI Dashboard** for analyzing and monitoring the supply chain operations of vegetable export and import businesses.

The dashboard provides actionable insights into inventory, warehouse performance, transportation costs, shipment status, and export trends, helping businesses make informed, data-driven decisions.

---

## 🚀 Dashboard Features

- 📊 Executive KPI Dashboard
- 📦 Inventory Performance Analysis
- 🏭 Warehouse Efficiency Monitoring
- 🚚 Transportation Cost Analysis
- 🌍 Region-wise Performance Analysis
- 🥬 Product Category Analysis
- 📈 Export Volume Trend
- 📦 Stock Level Monitoring
- 🚛 Shipment Status Tracking
- 🎯 Interactive Filters (Region & Category)

---

## 📌 Key Performance Indicators (KPIs)

| KPI | Description |
|-----|-------------|
| Warehouse Efficiency | Measures warehouse operational efficiency |
| Inventory Holding Days | Average number of days inventory remains in storage |
| Inventory Turnover Ratio | Inventory movement efficiency |
| Transportation Cost | Total transportation expenses |
| Export Volume | Total exported quantity |
| Average Transit Time | Average shipment delivery time |
| Shipment Status | Fulfilled, Pending, and Cancelled Orders |
| Stock Level | Current stock by category and region |
